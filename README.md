# 🛍️ Digi Tools – Digital Products Marketplace

## 🚀 Project Overview

**Digi Tools** is a modern and fully responsive web application where users can browse, search, and manage digital products easily. The platform focuses on delivering a smooth user experience with clean UI, efficient state management, and essential eCommerce-like features.

---

## ✨ Key Features

### 🔍 Product Browsing & Search

* View all available products in a clean grid layout
* Search products by **name and category**
* Real-time filtering for better UX

---

### 🛒 Wishlist Management

* Add products to wishlist
* Prevent duplicate items
* Remove items from wishlist
* Wishlist data stored in **localStorage (persistent)**

---

### 📦 Product Details Page

* Detailed view of each product
* Shows:

  * Name
  * Price
  * Category
  * Material
  * Dimensions
  * Stock status
  * Description

---

### 📊 Wishlist Summary

* Total number of items
* Total price calculation
* Dynamic shipping fee calculation
* Sticky sidebar for better UX

---

### 🔄 Sorting System

* Sort wishlist items by price:

  * High → Low
  * Low → High

---

### ❌ Empty State UI

* সুন্দর empty wishlist section
* User-friendly message when no items are added

---

### ⚡ Performance & UX

* Efficient state management with React Hooks
* Instant UI updates (no reload needed)
* Clean and responsive layout using Tailwind CSS

---

### 📱 Responsive Design

* Fully responsive for:

  * Mobile
  * Tablet
  * Desktop

---

### 🚫 Error Handling & Safety

* Safe JSON parsing from localStorage
* Prevents crashes when data is missing
* Duplicate product prevention logic

---

## 🛠️ Technologies Used

* **React.js**
* **React Router**
* **Tailwind CSS**
* **LocalStorage API**
